# bowenxiu.github.io
我的世界本是一片荒芜，直到你踩着月色而来
## [寂静的旷野](https://www.sweetpeach.cloud)

在长久逝去的岁月里，我的世界是一片望不到边际的荒芜。没有风吹过草叶的沙沙声，没有溪流穿过乱石的叮咚作响，甚至连时间的流动都变得极其缓慢而模糊。我就像是一个孤独的旅人，徒步在一片由灰色砂砾与枯萎枝桠构成的废墟中。

这里的夜漫长得没有尽头，墨色的天空沉重地压在头顶，连一丝微弱的星光都不曾施舍。我习惯了这种冷冽与寂静，以为生命本该就是这样一言不发的苍白。

---

## [破晓前的月色](https://www.darkspill.cloud )

打破这片死寂的，是一束毫无预兆倾泻而下的月光。

那不是刺眼的白昼之光，而是一种温柔到骨子里的皎洁。天空的阴霾被一只无形的手轻轻拨开，银辉如水，瞬间流淌过那些干涸的河床与长满青苔的乱石。也正是在那一刻，我听到了细碎的、不属于这片荒原的脚步声。

> 你就这样踩着一地月色，涉过岁月的荒凉，不急不徐地朝我走来。你的身影被月光拉长，每一步落下，都像是琴键上跳跃的音符，惊醒了沉睡在大地深处的某种回响。

---

## [盛开的绿洲](https://www.dayarena.cloud)

直到你站到我的面前，我才真正看清了世界的颜色。

你带来的不仅仅是光，还有万物的复苏。你踩过的沙砾开始长出嫩绿的新芽，你呼吸过的空气里多了一丝泥土与花朵的芬芳。那一刻，我听见冰川消融的声音，看见繁星如雨般坠入我原本空洞的眼眸。

有的人来到你的生命里，是为了和你一起看风景；而你来到我的世界，却是为了亲手替我栽种出一片绿洲。从此，荒芜退场，四季轮转，我的宇宙因你而有了白昼与歌声。
